# 🌤️ App Clima

App de previsão do tempo que busca dados reais de qualquer cidade do mundo.

## 🚀 Demo

[🔗 Acesse o projeto aqui](https://lucasimoes6.github.io/app-clima)

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
- OpenWeather API

## ⚙️ Como usar

1. Digite o nome de uma cidade
2. Clique em **Buscar**
3. Veja a temperatura e condição do tempo
